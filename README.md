# Visual Memory Game

A desktop visual memory game built with `pygame`.

## Setup

```bash
python3 -m pip install --user pygame
```

## Run

```bash
python3 memory_game.py
```

## How to play

1. Press `Start Game`.
2. Watch the highlighted tile pattern.
3. Click the tiles back in the same order.
4. Each successful round adds one more tile.

## Notes

- Tested with Python 3.9 and `pygame 2.6.1`.
- If `python3` cannot find `pygame`, make sure the same Python installation was used for both install and run.
